Canvas Instructions for BIHS Courses

Learn how to access and manage your BIHS courses on Canvas.

Accessing Videos for the Summer Discovery Series

Videos for the Summer Discovery Series and the Conceptual Study of Srimad Bhagavatam are hosted on Canvas Instructure, a free and secure platform. To ensure a more interactive and academic experience, we use Canvas to keep the conversation going and strengthen networking within an academic setting.

Welcome to the Canvas Guide

This guide will walk you through logging in to Canvas, creating accounts, and adding the Summer Discovery Series and Conceptual Study of Srimad Bhagavatam courses.

1. Log in to Canvas

If you already have a Canvas account, simply log in with your email and password.

  • Enter your email and password.
  • Click "Log in" to access your dashboard.
Log in to Canvas

2. Create Account and Add Discovery Series

If you're new to Canvas, you'll need to create a free student account. After creating your account, use the following code to add the Summer Discovery Series:

  • Click the link below to sign up for a free account.
  • Use the code L6FWRW to add the Discovery Series to your account.
Create Account

3. Create Account and Add Conceptual Study of Srimad Bhagavatam

If you're new to Canvas, you'll need to create a free student account. After creating your account, use the following code to add the Conceptual Study of Srimad Bhagavatam:

  • Click the link below to sign up for a free account.
  • Use the code L6H6BG to add the Srimad Bhagavatam course to your account.
Create Account

4. Add Conceptual Study of Srimad Bhagavatam to Discovery Series

If you're enrolled in the Summer Discovery Series and would like to add the Conceptual Study of Srimad Bhagavatam, follow these steps:

  • Log in to Canvas and open your Discovery Series course.
  • Use the link below to add the Srimad Bhagavatam course to your list.
  • Use the code L6H6BG to add the Srimad Bhagavatam course.
Add Conceptual Study
